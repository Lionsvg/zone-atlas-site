Zone Atlas — site simple (statique)
-----------------------------------
Fichiers :
- index.html : page d'accueil avec bouton de téléchargement
- instructions.html : guide d'installation du launcher
- css/styles.css : styles
- assets/ZoneAtlasLauncher.zip : archive factice à remplacer par votre vrai launcher

Remplacez le fichier assets/ZoneAtlasLauncher.zip par votre installateur réel
(ou modifiez le lien dans index.html pour pointer vers l'URL de téléchargement).
